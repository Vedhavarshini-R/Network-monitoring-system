# Network Monitoring System

## Overview
Network Monitoring System is a full-stack web application designed to monitor, analyze, and manage network signal performance in real time.

This system helps users track signal strength, analyze network performance, generate reports, and monitor service activity through an interactive dashboard.

## Features
- Real-time network signal monitoring
- Dashboard with analytics
- Activity logs tracking
- Notifications management
- Reports generation
- Performance analysis
- User authentication and secure access
- Service management
- Settings configuration

## Tech Stack

### Frontend
- Angular
- TypeScript
- HTML
- CSS

### Backend
- ASP.NET Core
- C#
- Entity Framework Core

### Database
- PostgreSQL

## Project Structure
```bash
NetworkMonitoringSystem/

```

## Installation

### Frontend
```bash
cd frontend
npm install
ng serve
```

### Backend
```bash
cd backend
dotnet restore
dotnet run
```
